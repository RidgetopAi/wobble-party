# Test Audio Fixtures

These short WAV files are synthetic probes for Wobble Brain tuning.

- `kick-pulse.wav`: low-frequency pulses every 0.5s for bass/kick behavior.
- `steady-pad.wav`: sustained midrange tone stack for smoothing and false-positive checks.
- `high-ticks.wav`: short high-frequency ticks for brightness/high-band behavior.
- `busy-mix.wav`: mixed low pulses, mid tone, and high ticks for overlap behavior.

See `calibration/README.md` for known-frequency measurement fixtures.

They are intentionally simple. The goal is repeatability, not musical quality.
